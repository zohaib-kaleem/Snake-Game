# 🐍 Classic Snake Game (Web Edition)

A sleek, responsive, and browser-based implementation of the classic Snake Game. Built using vanilla web technologies, this project focuses on clean DOM manipulation and JavaScript game loops.


---

## 🚀 Technologies Used

* **HTML5:** Structure and Canvas API for rendering the game grid.
* **CSS3:** Styling the game board, typography, and responsive layout.
* **JavaScript (ES6):** Core game logic, collision detection, and score management.

## 🛠️ Features

* **Responsive Design:** Playable on various screen sizes.
* **Keyboard Support:** Control the snake using Arrow keys or `WASD`.
* **Dynamic Scoring:** Score increases as the snake consumes food.
* **High Score Memory:** Uses `localStorage` to keep track of your best run.
* **Clean UI:** Minimalist aesthetic for a focused gaming experience.

## 📂 File Structure

| File | Description |
| --- | --- |
| `index.html` | The main entry point and game layout. |
| `style.css` | Definitions for the game board, colors, and animations. |
| `script.js` | The "brain" of the game — handles movement and logic. |

## 🧠 Game Logic Overview

The game operates on a fixed-interval loop where the following steps occur every few milliseconds:

1. **Clear:** The canvas is wiped clean.
2. **Move:** The snake's coordinates are updated based on the current direction.
3. **Check:** The engine checks for collisions with walls, self, or food.
4. **Draw:** The snake and food are re-rendered at their new positions.
